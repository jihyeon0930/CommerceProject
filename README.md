# 🛒 실시간 커머스 플랫폼 (Java)

## 📌 프로젝트 개요
이 프로젝트는 Java 콘솔 환경에서 동작하는 **실시간 커머스 플랫폼**입니다.  
사용자와 관리자의 기능을 **상태(State) 기반**으로 구현하여, 화면 전환과 기능 흐름을 관리합니다.

---

## ⚙️ 기능

### 👤 사용자 기능
- 🗂️ 카테고리별 상품 목록 확인
- 🔍 상품 상세 정보 확인
- 🛒 장바구니 담기 / 주문
- ❌ 주문 취소

### 🛡️ 관리자 기능
- 🔑 관리자 인증 (비밀번호)
- ➕ 상품 추가 / 수정 / 삭제 (!미구현)
- 📦 재고 관리 (!미구현)
- 📊 전체 상품 현황 (!미구현)

---

## 🏗️ 시스템 구조

### 🔄 State 기반 화면 전환
- Enum `State`를 사용하여 메인, 카테고리, 상품 상세, 장바구니, 주문, 관리자 화면 상태 관리

### 📂 클래스 구조
- `CommerceSystem` : 사용자 기능 전담
- `AdminSystem` : 관리자 기능 전담
- `Category` / `Product` : 데이터 모델
- `Utils` : 입력 처리 공용 메서드

### 🔗 데이터 공유
- `categories` 리스트를 `CommerceSystem`과 `AdminSystem`에서 참조하여 실시간 데이터 공유
- `Scanner` 또한 공유하여 입력 일관성 유지

---

## 💡 주요 구현 포인트
1. 🔄 상태(State) 기반 메뉴 구조
2. 🛡️ AdminSystem 분리로 사용자/관리자 기능 명확히 구분
3. 🔧 입력 처리 공용 메서드(`Utils.getIntInput`)로 중복 제거
4. 🔁 while + return 조합으로 반복 입력 처리 및 화면 전환
5. 📦 재고, 장바구니, 주문 관리 모두 실시간 데이터 반영 (!미구현)
